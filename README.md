# 🎙️ VoiceLedger - 语音记账本

一款 Material Design 3 风格的安卓记账应用，支持语音输入自动识别金额和分类。

## ✨ 功能特性

### 📝 手动记账
- 支持收入/支出记录
- 选择分类：餐饮、交通、购物、娱乐、住房、医疗、教育、其他
- 填写金额、备注、日期

### 🎤 语音记账
- 点击麦克风按钮，说出"今天午餐花了35块"
- 自动识别金额（35）、分类（餐饮）
- 支持的语音关键词：
  - **餐饮**：吃、饭、外卖、奶茶、咖啡、火锅...
  - **交通**：打车、地铁、公交、加油、机票...
  - **购物**：买、淘宝、京东、衣服、超市...
  - **娱乐**：电影、游戏、KTV、旅游...
- 识别完成后弹出确认框，可手动修改再保存

### 📋 账单列表
- 按时间倒序显示所有记录
- 卡片式设计，显示分类图标、金额、备注、日期
- 点击删除按钮确认后删除记录

### 📊 月度统计
- 顶部显示本月总支出/总收入
- 饼图展示各分类支出占比
- 下方列出各分类详细金额和百分比

### 📤 数据导出
- 一键导出本月账单为 CSV 文件
- 可分享到微信、邮件或保存到本地
- CSV 格式兼容 Excel 打开

## 🔒 隐私保护
- 所有数据本地存储（Room 数据库）
- 不联网，不收集任何个人信息
- 语音识别使用 Android 原生 SpeechRecognizer

## 🛠️ 技术栈
- **语言**：Kotlin
- **最低版本**：Android 8.0 (API 26)
- **数据库**：Room
- **图表**：MPAndroidChart
- **视图绑定**：ViewBinding
- **设计**：Material Design 3
- **架构**：MVVM + Repository

## 📦 安装方式

### 方式一：下载编译好的 APK
1. 前往 [Releases](../../releases) 页面下载最新 APK
2. 或在 [Actions](../../actions) 页面下载构建产物

### 方式二：自行编译
```bash
git clone https://github.com/YOUR_USERNAME/VoiceLedger.git
cd VoiceLedger
./gradlew assembleDebug
```
编译后的 APK 在 `app/build/outputs/apk/debug/` 目录下。

## 📱 使用说明

1. **添加账单**：点击底部"记账" Tab → 填写信息 → 点击"保存"
2. **语音记账**：点击底部"记账" Tab → 点击麦克风按钮 → 说话 → 确认 → 保存
3. **查看账单**：点击底部"账单" Tab 查看所有记录
4. **查看统计**：点击底部"统计" Tab 查看月度数据
5. **导出数据**：在统计页点击"导出本月账单"按钮
